# 1.2. Módulo Artefato Generalista

## 5W2H

A equipe decidiu utilizar o artefato 5W2H, uma ferramenta simples e objetiva que pode ser aplicada ao planejamento e à gestão de projetos. O 5W2H é composto por sete etapas, estruturadas em perguntas que ajudam a elicitar aspectos importantes do projeto [[1]](#ref1). São elas:

| Grupo | Itens |
| --- | --- |
| 5W | - **What?** O que será feito (Etapas)<br> - **Why?** Porque será feito (Justificativa)<br> - **Who?** Quem executará a tarefa (Responsabilidade)<br> - **When?** Quando será feito (Tempo)<br> - **Where?** Onde será feito (Local) |
| 2H | - **How?** Como será realizada a tarefa (Método)<br> - **How much?** Quanto custará para ser feito (Custo) |

Fonte da tabela: [[1]](#ref1).

Cada membro da equipe colaborou com sua visão do projeto por meio desse artefato. Para padronizar os registros, a equipe deve utilizar o template a seguir.

### Template da Equipe (Preenchimento Individual)


| ID do artefato | Código único do registro no padrão **#5W2HNN**. |
|--- | --- |
| **Autor** | Nome de quem elaborou o 5W2H. |
| **What?** | Entrega, ação ou etapa que será realizada. |
| **Why?** | Justificativa e objetivo do que será feito. |
| **Who?** | Responsável principal pela execução. |
| **When?** | Prazo, data ou período de realização. |
| **Where?** | Local, ambiente ou contexto em que a ação ocorrerá. |
| **How?** | Método, processo, ferramentas ou abordagem utilizada. |
| **How much?** | Custo estimado (valor, esforço ou recursos). |

---
### Artefatos gerados pela equipe: 

| ID do artefato | **#5W2H01**. |
|--- | --- |
| **Autor** | Heloisa |
| **What?** | A equipe desenvolverá um sistema web em formato de Blog para conectar amantes de trilhas e natureza que tenham interesse em conhecer belezas naturais em suas redondezas ou em novos lugares. |
| **Why?** | O objetivo é disponibilizar uma plataforma que conecte pessoas que se conheçam trilhas e ofereçam caravanas e excursões para conhecer os locais compartilhados dentro de sistema. |
| **Who?** | A equipe será responsável pela realização. |
| **When?** | O projeto será realizado durante o semestre, sendo o final previsto para 29/06 |
| **Where?** | No ambiente da faculdade e nas casas dos estudantes |
| **How?** | A equipe decidiu utilizar o Scrum, combinado com Kanban |
| **How much?** | Considerando o salário médio de um profissional Junior como 2600 mensais e que o projeto custará por volta de 60 horas trabalhadas, o custo sairia por volta de 7818 reais |
